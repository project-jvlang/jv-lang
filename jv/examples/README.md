# JV言語 サンプルコード集

このディレクトリにはJV言語の様々な機能を実演するサンプルプログラムが含まれています。

## 動作確認済みサンプル

1. **02-variables.jv** - 型推論と変数宣言
   - `val`による不変変数
   - `var`による可変変数
   - 型推論
   - 文字列補間

2. **03-functions.jv** - 関数の定義と呼び出し
   - 型注釈付き関数宣言
   - 戻り値型
   - 関数呼び出し

3. **05-string-interpolation.jv** - 文字列補間機能
   - `${variable}`による変数補間
   - `${expression}`による式補間

4. **06-basic-types.jv** - 基本データ型
   - 整数型
   - 文字列型
   - 真偽値型
   - Double型（型注釈が必要）

5. **07-arithmetic.jv** - 算術演算
   - 整数演算（+, -, *, /, %）
   - 浮動小数点演算
   - 数値演算用の型注釈

## サンプルの実行方法

サンプルを実行するには：

```bash
jv/target/debug/jv run jv/examples/<ファイル名>.jv
```

例：

```bash
jv/target/debug/jv run jv/examples/02-variables.jv
```
